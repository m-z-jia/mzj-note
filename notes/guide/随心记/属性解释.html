# 随心记属性解释

本文档详细解释随心记应用的各项配置属性，帮助用户理解和自定义应用功能。

# 基本设置

# 常规

  • 自动保存：启用后，编辑内容会自动保存，无需手动点击保存按钮
  • 启动时打开上次文件：应用启动时自动打开上次关闭时的文件
  • 新建文件默认路径：设置新建文件的默认保存位置
  • 默认文件扩展名：设置新建文件的默认扩展名，如 .md、.txt 等
  • 自动换行：启用后，文本会根据窗口宽度自动换行
  • 显示行号：在编辑器左侧显示行号
  • 显示空白字符：显示空格、制表符等空白字符
  • 显示行尾符：显示行尾的换行符
  • 启用拼写检查：启用拼写检查功能，标记可能的拼写错误

# 编辑器

  • 字体：设置编辑器字体
  • 字体大小：设置编辑器字体大小
  • 行高：设置编辑器行高
  • Tab 宽度：设置 Tab 键的缩进宽度
  • 使用空格代替 Tab：启用后，按 Tab 键时插入空格而非 Tab 字符
  • 缩进指南：显示代码缩进的辅助线
  • 高亮当前行：高亮显示当前编辑行
  • 高亮匹配括号：当光标位于括号附近时，高亮显示匹配的括号

# 界面设置

# 外观

  • 主题：选择应用主题，如浅色、深色等
  • 图标主题：选择应用图标主题
  • 工具栏：设置工具栏的显示方式和内容
  • 状态栏：设置状态栏的显示内容
  • 侧边栏：设置侧边栏的显示方式和内容
  • 标签栏：设置标签栏的显示方式

# 布局

  • 编辑器布局：设置编辑器的布局方式，如单列、双列等
  • 预览布局：设置预览窗口的布局方式
  • 面板位置：设置各种面板的位置
  • 分割比例：设置分割窗口的比例
